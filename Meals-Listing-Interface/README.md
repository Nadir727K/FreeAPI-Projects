##  Description

This project is a React-based web application that displays a list of meals using a public API. The objective of this project is to understand how to fetch and handle API data, and present it in a structured, visually clear, and responsive user interface.

The application retrieves meal data and displays it in a card-based layout, making it easy to browse different meals along with their categories and origin.

---

##  Features

* Fetches meals data from a public API
* Displays meals in a responsive grid layout
* Shows meal image, name, category, and area (origin)
* Loading and error handling states
* Clean and modern UI using CSS

---

##  Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS (custom, no frameworks)

##  Installation & Setup

1. Clone the repository:

```bash
git clone <your-repo-link>
```

2. Navigate to the project:

```bash
cd Meals-Listing-Interface
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
