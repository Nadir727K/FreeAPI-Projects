# Random Users React App

## 📌 Description

This project is a React-based web application that fetches and displays user data from a public API. The goal of this project was to understand how to work with APIs, handle asynchronous data, and present the data in a clean and structured user interface.

The application retrieves random users and displays them in a responsive card layout showing basic profile information.

## 🚀 Features

* Fetches user data from a public API
* Displays users in a responsive grid layout
* Shows user profile image, name, email, and location
* Loading and error handling states
* Clean and minimal UI using plain CSS

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone <your-repo-link>
```

2. Navigate to the project:

```
cd random-users-app
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
